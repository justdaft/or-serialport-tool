# or-serialport-tool
A simple serial port debug tool based on electron&amp;vue.


## why?

I can't find a _good enough_ serial port debug tool in macOS, when the baud rate is **1M**(1024000).


## Tech

  - electron 1.4.x
  - vue 2.x
  - webpack 2.2.0


it's a cross-platform tool.

## Usage

    // run dev mode
    npm install

    // run dev mode
    npm run dev

    // run pack mode
    npm run pack

    // create package
    npm run package
